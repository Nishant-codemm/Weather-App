# 🌦️ Weather App

A modern and responsive Weather App built using HTML, CSS, and JavaScript that fetches real-time weather data using an API.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Real-time temperature display
* 🌥️ Dynamic weather icons (Clouds, Rain, Clear, etc.)
* 💨 Wind speed and 💧 humidity info
* ⚠️ Error handling for invalid cities
* 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (Vanilla JS)
* OpenWeatherMap API

## ⚙️ How to Run Locally

1. Clone the repository

```
git clone https://github.com/Nishant-codemm/Weather-App.git
```

2. Open `index.html` in your browser

---

## 🔑 API Setup

1. Go to OpenWeatherMap
2. Generate your API key
3. Replace the API key in `script.js`

```javascript
const apiKey = "YOUR_API_KEY";
```
